The Matrix is a next-generation cybersecurity platform that provides comprehensive network protection, social safety monitoring, cyber training, and community-driven threat intelligence.

Built with enterprise-grade security in mind, The Matrix combines multiple cybersecurity modules into a unified interface with real-time monitoring and automated response capabilities.

üöÄ Getting Started

System Requirements

  • Operating System: Windows 10/11, Linux (Ubuntu 20.04+), macOS 11+
  • RAM: Minimum 8GB (16GB recommended)
  • Storage: 500GB available space
  • Network: Stable internet connection
  • Browser: Chrome 90+, Firefox 88+, Edge 90+

Quick Start

  1. Access the platform at http://localhost:5500
  2. Click "Enter System" to access the main dashboard
  3. Select a module from the homepage
  4. Configure your security settings
  5. Start monitoring your network

GET /api/threat-score - Get current threat score

GET /api/flows - Retrieve network flows

üèóÔ∏è Architecture

System Components

  • Frontend: HTML/CSS/JavaScript (Matrix theme)
  • Backend: Python FastAPI microservices
  • Agent: Python + Rust network monitor
  • Database: PostgreSQL + Redis
  • Message Queue: Redis Streams / Kafka
  • Logs: Loki + ClickHouse

Data Flow

  1. Agent captures network packets (dumpcap)
  2. Zeek/Suricata analyze traffic
  3. Logs sent to backend via WebSocket/REST
  4. Correlation engine processes alerts
  5. Threat score computed (AI + rules)
  6. Dashboard displays real-time data
  7. Auto-blocking triggered if needed

üîç Troubleshooting

Common Issues

Q: Dashboard not loading?

A: Ensure Live Server is running and you're accessing the correct port (5500)

Q: Matrix rain not showing?

A: Check browser console for JavaScript errors. Refresh the page.

Q: Backend not connecting?

A: Verify backend is running on port 8000. Check CORS settings.

Q: Agent not registering?

A: Ensure agent has network permissions and correct API endpoint.
